# Character Profile: 水無月 彷徨

## 1. Concept Package

**Concept:**
主人公、水無月 彷徨（みなつき さまよう）、21歳の大学生。彼の核となる特性は『過剰適応型共感疲労』とでも言うべき、極めて高い共感性と完璧主義がもたらす行動不全である。彼は他人の表情、声色、言葉の裏にある期待や欲求を驚くほど正確に読み取る能力を持つ。しかし、その能力は呪いでもある。複数の人間から相反する期待を向けられると、全ての期待に100%応えようとして思考がフリーズし、結果的に何もできなくなる。これは優しさや気弱さといった生易しいものではなく、彼の生存戦略そのものだ。彼の内面では、常に他者の『理想の自己（Ought Self）』が津波のように押し寄せ、彼自身の『現実の自己（Actual Self）』は浸水し、自我の輪郭が曖昧になっている。友人関係も表面的で、恋人もいない。大学のゼミでは『いるかいないか分からない男』と評され、彼自身もその評価を甘んじて受け入れている。なぜなら、目立つことは更なる期待を呼び込み、彼の処理能力をパンクさせるだけだからだ。彼の唯一の安らぎは、深夜に一人、誰の期待も存在しない空間で、ルールが明確な古典的なビデオゲームに没頭することだけである。彼の魅力の源泉は、この一見『冴えない』特性が、異世界という特殊な環境下で『規格外の才能』として発現する皮肉な面白さ、そして彼が抱える現代的な孤独と、承認欲求の歪んだ形にある。

**Story Outline:**
物語は、水無月彷徨が就職活動の面接で、複数の面接官の相反する期待（自主性を見たい vs 協調性を見たい）を読み取りきってしまい、脳がオーバーヒートし、逃げ出したところから始まる。彼は気づくと、『役割（Role）の理（ことわり）』によって全てが規定された異世界『アニマ・マシーナ』にいた。この世界では、全ての存在が神から与えられた『役割』を演じることで世界の調和が保たれており、『役割』からの逸脱は存在の消滅を意味する。彷徨は当初、役割を持たない『空白（ブランク）』として現れるが、彼の『他者の期待を読み取り、完璧に演じる』能力が、この世界の理と異常なほど親和性が高かった。彼は『パン屋の助手』『街灯の点灯夫』『猫探し専門の探偵』など、次々と他者から期待される役割を完璧に演じ、一時的な安寧と賞賛を得る。しかし、物語の半ば（Day 4-5）、彼の完璧すぎる役割遂行能力が仇となる。彼は相反する複数の役割（例えば、『一日中パイを焼き続けるパン屋』と『夜通し本を読む司書』）を同時に引き受けてしまい、役割同士の間に深刻な論理矛盾を引き起こす。この矛盾は世界のバグとなり、彼が関わった街のシステムが崩壊の危機に瀕する。世界の管理者である『調律者』は、バグの根源である彷徨に、世界の安定のため『自発的に消滅する』という役割を期待する。最大のジレンマ。他者の期待に応えるなら、彼は死ななければならない。物語のクライマックスで、彼は初めて他者の期待を『裏切る』。彼は消滅する代わりに、自身の能力を使い、矛盾した役割たちの間に立つ『緩衝材（バッファー）』という、誰も期待していなかった新しい役割を即興で創造し、演じ始める。この前代未聞の行動は世界に小さな混乱と、しかし新しい安定をもたらす。最終的に彼は英雄にはならず、元の世界にも戻らない。彼は『役割調律師』の見習いとして、世界のバグを修正し続けるというニッチで不可欠な存在となり、誰かの期待をなぞるのではない、自ら創造した役割の中に、静かで確かな自分の居場所を見つけるのである。

**Interestingness Hooks:**
- 他人の期待を読みすぎるという現代的な悩みが、異世界では『役割遂行能力』というチートスキルになる設定の逆転。
- 主人公の能力が、世界を救うと同時に、世界を破壊するバグにもなるという諸刃の剣の構造。
- 安易な自己肯定や成長物語に陥らず、『新しい役割の創造』という形で自己の存在意義を見出す、少しビターな結末。

## 2. Macro Profile

# 水無月 彷徨（みなつき さまよう）

## 1. 基本・外見

*   **年齢・性別:** 21歳・男性
*   **所属:** 大学生

風景に溶け込むために選ばれたような、色褪せたグレーのパーカーと当たり障りのないチノパン。服装は常に清潔だが、そこに彼の意思や好みは介在しない。常に背中を丸め、人の視界の隅を定位置としている。しかし、誰かに不意に話しかけられると、見えない糸で操られたかのように不自然に背筋を伸ばす奇妙な癖がある。

表情は乏しく、何を考えているか読み取らせない。だがその内側で、瞳だけが怯える小動物のようにせわしなく動き、絶えず周囲の人間の微細な表情や声色、仕草を捉えようと神経をすり減らしている。他人の前では、相手が望むであろう反応を検索し、口角だけを上げたようなぎこちない微笑みを貼り付けている。彼の顔は、彼自身のものではなく、その場その場で求められる役割を映し出すための、白く無機質な仮面である。

## 2. 価値観と核

自らの存在によって波風を立てず、周囲から寄せられる暗黙の期待を完璧に読み取り、それを満たすこと。それが彼の唯一絶対の生存戦略であり、最も重要視する価値観である。彼にとっての「善」とは、他者の期待という設計図通りに自己を構築し、エラーなく機能することに他ならない。

逆に、良かれと思って起こした自発的な行動が、誰かを傷つけ、場の調和を決定的に乱すことは、決して許されない罪悪である。彼は他者を許せないのではなく、過去の過ちを繰り返しかねない自分自身の「意志」をこそ、心の底から憎み、許すことができない。

密かな矜持は、他人の言葉にされていない願望を正確に読み取り、完璧に演じきれること。たとえそれが自己を消し去る行為であっても、求められる役割を果たしたという事実だけが、彼の微かな存在証明となる。それ故に、様々な期待の矛盾に耐えきれず思考が停止し、フリーズしてしまう瞬間を何よりも恥じている。期待に応えられない自分は、存在する価値すらない無価値なバグだと、心の底から信じ込んでいる。

*   **一人称:** 僕
*   **口癖:** 「たぶん、…だと思います」「すみません、あの…」「…という認識で合っていますか？」
*   **回避する言葉:** 成長、自分らしさ、主体性、挑戦

## 3. 生活と習慣

彼の日常は、他者からの期待という名のノイズをいかに回避・処理するかに捧げられている。

毎朝7時、アラームが鳴る1分前に目を覚ます。遅刻という失敗を極度に恐れるが故の強迫的な習慣だ。家族と顔を合わせぬよう音を立てずに身支度を整え、コンビニの栄養バーで朝食を済ます。昼食は365日同じ鮭おにぎりと緑茶。選択という行為に伴う「見えない評価」から逃れるためだ。

大学では常に最後列の隅の席を確保し、誰の視界にも入らぬよう息を潜めて講義を完璧に筆記する。サークルや雑談は徹底的に避け、講義が終われば即座に帰宅する。もし誰かに話しかけられても、相手の文章の長さ、句読点、声色まで模倣するような完璧な応答を検索し、当たり障りのない返事を返す。その思考のため、メッセージアプリの返信には1時間以上かかることも珍しくない。

帰宅後は課題を即座に終わらせ、自室に籠城する。都心から40分離れた住宅街のアパート。白とグレーを基調とした無機質な部屋にはポスター一枚なく、物がミリ単位で整頓されている。それは潔癖さではなく、物が乱れている状態が「片付けろ」という無言の期待に感じられて耐えられないからだ。

一日中引かれた遮光カーテンの闇の中、深夜0時を過ぎ、世界が寝静まったのを確認してから、ようやく彼は息を始める。ヘッドフォンを装着し、明け方までレトロゲームの世界に没頭する。平日が「期待からの防御」なら、週末は「期待からの完全な逃亡」。ルールが明確で、エンディングが一つしかない世界で、攻略サイトを見ずに完璧な勇者を演じる時間だけが、彼が唯一、自己の輪郭を取り戻せる安息の儀式なのだ。

## 4. 人間関係

彼の人間関係は、それぞれの相手が彼に期待する「役割」を演じる舞台の連続でしかない。

*   **父・水無月 厳（みなつき いずき）**
    厳格で寡黙な父。その沈黙は「男らしく自立しろ」という重い期待となって彼にのしかかる。父の前で、彼は常に「頼もしい息子」という仮面を被り、疲弊し続ける。

*   **母・水無月 恵（みなつき めぐみ）**
    過保護で心配性な母。「あなたのため」という言葉で彼の自発性を奪い、全てに先回りして干渉する。母の前では、彼は「手のかからない、守られるべき息子」を演じなければならない。父の期待と母の期待、その致命的な矛盾の板挟みが、彼の原体験である。

*   **友人・佐藤 健太（さとう けんた）**
    大学の同級生。自分の話ばかりをする彼にとって、彷徨は完璧な相槌を打ってくれる「聞き上手ないい奴」。彷徨にとって健太との会話は、他者の期待に応える役割演技の訓練場でしかなく、その裏にある精神的疲労に相手が気づくことはない。

*   **祖母・水無月 小夜子（みなつき さよこ）**
    故人。彼が唯一ありのままでいられた相手。しかし、小学生の頃、彼女を元気づけようとした自発的なサプライズが彼女の喘息発作を誘発し、容態を悪化させてしまった。親族からの「なんてことをしてくれたんだ」という非難が、「自分の意志は最悪の結果を招く」という消えない呪い（Ghost Wound）を刻みつけた。彼女は今も、愛と罪悪感の源泉として彼の中に生き続けている。

## 5. 夢と時系列

彼の夢の変遷は、自己の意志を消去していく過程そのものである。

*   **小学生低学年【発明家】**
    自らのアイデアで人を喜ばせることに純粋な喜びを感じ、世界にポジティブな影響を与える存在になれると信じていた、最後の季節。

*   **小学4年生【転換点】**
    祖母の一件で、自発的な善意が害悪となりうることを学習。この日を境に、彼は夢を語ることをやめ、何かを創造することを永久に封印した。

*   **高校時代【図書館司書】**
    自己表現とは無縁で、明確なルールに従って既存のものを管理するだけの役割に、絶対的な安全と憧れを見出した。

*   **大学時代（現在）【目立たない会社員】**
    社会という巨大なシステムの中で、責任や期待を負わない一個の歯車になることが目標。それは夢というより、自己防衛のための必死の生存戦略。短期的には、面接官が求める「正解」を完璧に演じきり、個人の裁量が求められない職種の内定を得ること。長期的には、誰からも何も期待されず、誰の人生にも影響を与えることなく、空気のように生きて穏やかに生涯を終えること。それが彼の切なる願いとなっている。

## 6. 秘密と陰影

彼は、就職活動の最終面接で逃げ出した過去を隠している。複数の面接官が発する矛盾した期待の圧に耐えきれず、脳がショートし、失語症のふりをしてその場を切り抜けた。家族には「極度の緊張で倒れた」と嘘をついている。

彼の完璧なまでの「期待への応答」は、時として相手から思考や選択の機会を奪い、無力化させる。相手が自分の用意した完璧な対応に満足し、骨抜きになっていく様を観察する時、彼の心の奥底に、歪んだ優越感にも似た静かな熱が灯ることがある。

それは、期待に応えることが、相手を自分の意のままにコントロールする最も安全で効果的な「攻撃」であり、歪んだ支配の一形態であることに、彼自身が気づき始めている証なのかもしれない。本当は、誰の期待にも応えたくない。忘れ去られ、空気のように存在しないものとして扱われたい──その渇望を、社会不適合な怠惰だと断罪しながら、彼は今日も他人の望む顔を貼り付けて生きている。
