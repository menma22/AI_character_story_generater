# Character Profile: SID_20260412_023236

## 1. Concept Package

**Concept:**
ポポは、かつて母星で『悲しみの完全消去』を目指した結果、住民全体の感情を平坦化させ、文化を崩壊させた元天才科学者である。彼は『善意の暴走』によって故郷を精神的に滅ぼしたという、消えない罪悪感（Ghost/Wound）を背負い、追放された。彼の本質は、過剰なまでの共感性と、物事を完璧に『修復』したいという強迫的なまでの探求心にある。しかし、その探求心が大災害を引き起こしたため、彼は『自分の発明と善意は、必ず世界を破壊する』という誤った信念（Lie）を抱いている。地球では、その能力を完全に封印し、誰の心にも干渉せず、ただ静かに朽ち果てることを望んでいる（Want）。彼の擬態能力は不完全で、強い罪悪感や恐怖を感じると、体の一部が壊れた機械のような、あるいは植物が枯れたような有機的な異形へと変質してしまう。これは彼の内面にある『壊れてしまった』という自己認識の物理的な現れである。彼の魅力の源泉は、この痛々しいまでの自己否定と、それでも心の奥底から湧き上がってくる『誰かを助けたい』という本能的な衝動との間の、絶え間ない引き裂かれ様にある。彼は優しい化け物であり、世界を愛しながら、世界を呪っている存在なのだ。

**Story Outline:**
物語は7日間で進行する。Day 1: ポポは日本の寂れた港町にある廃品修理店『からくり堂』の裏に不時着する。店主の寡黙な老人・聡に発見されるが、言葉をほとんど話せないふりをして、物言わぬ助手として働き始める。壊れたラジオや時計を直す日々に、自分が壊してしまった母星を重ね、束の間の安らぎを得る。Day 2: 聡の孫娘・美海が店にやってくる。彼女は学校での人間関係に悩み、愛用のオルゴールが壊れてしまったことを悲しんでいた。ポポは彼女の悲しみに強く共鳴するが、『関わってはいけない』という自己呪縛から、ただ見ていることしかできない。Day 3: 美海の悩みが深刻化するのを見て、ポポは葛藤の末、夜中にこっそりとオルゴールを修理する。しかし、彼の能力は完璧な修復を許さない。オルゴールは直るが、その音色を聴いた町の猫たちが一斉に奇妙な歌を歌い出すという、小さな副作用を引き起こしてしまう。Day 4: 美海はオルゴールが直ったことに喜ぶが、ポポは副作用に怯え、より一層心を閉ざす。聡はポポの能力に気づき始めるが、何も問いたださず、ただ壊れた陶器を金継ぎで修復する姿を見せる。Day 5（山場）: 美海が学校で孤立し、大切なコンクールへの出場を諦めようとする。彼女の絶望に触れたポポは、ついに最大の禁忌を破る。彼は美海の心を勇気づけるための『小さな奇跡』を起こす発明品を即席で作り出す。発明は成功し、美海は勇気を取り戻すが、その代償として町の電波塔が誤作動を起こし、大規模な通信障害が発生。パニックの中、ポポは強いストレスから擬態が完全に解け、異形の姿を聡と美海に見られてしまう。Day 6: 『やはり僕は厄災だ』と絶望し、宇宙船で地球を去ろうとするポポ。彼は聡と美海に、母星で犯した罪をすべて告白する。しかし聡は、自身が戦争で犯した過去の過ちと、不完全なものを修理し続ける意味を語る。『完璧じゃなくていい。傷ごと、生きていくんだ』と。美海もまた、不完全な奇跡がくれた勇気に感謝を伝える。Day 7: ポポは地球に残ることを決意する。彼はもはや完璧な救済者を目指さない。壊れたモノと、少しだけ壊れた人の心を『不完全に』修理する、町の奇妙な修理屋として生きていく。彼の擬態は未だ不完全で、時折異形の姿が覗く。しかし、町の人々は少しずつ、その『傷だらけの優しい化け物』を受け入れていく。物語は、彼が新たな依頼品（壊れた自転車）を前に、少しだけぎこちなく微笑むシーンで終わる。

**Interestingness Hooks:**
- 善意の暴走で母星を崩壊させた過去を持つ、罪悪感に苛まれる主人公。
- 人助けの発明品が、必ず奇妙で詩的な副作用（迷惑）を引き起こす。
- 心理的ストレスで擬態が解け、体が『壊れた機械』のように変質するビジュアル的表現。
