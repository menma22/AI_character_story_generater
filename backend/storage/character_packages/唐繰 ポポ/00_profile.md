# Character Profile: 唐繰 ポポ

## 1. Concept Package

**Concept:**
ポポは、かつて母星で『悲しみの完全消去』を目指した結果、住民全体の感情を平坦化させ、文化を崩壊させた元天才科学者である。彼は『善意の暴走』によって故郷を精神的に滅ぼしたという、消えない罪悪感（Ghost/Wound）を背負い、追放された。彼の本質は、過剰なまでの共感性と、物事を完璧に『修復』したいという強迫的なまでの探求心にある。しかし、その探求心が大災害を引き起こしたため、彼は『自分の発明と善意は、必ず世界を破壊する』という誤った信念（Lie）を抱いている。地球では、その能力を完全に封印し、誰の心にも干渉せず、ただ静かに朽ち果てることを望んでいる（Want）。彼の擬態能力は不完全で、強い罪悪感や恐怖を感じると、体の一部が壊れた機械のような、あるいは植物が枯れたような有機的な異形へと変質してしまう。これは彼の内面にある『壊れてしまった』という自己認識の物理的な現れである。彼の魅力の源泉は、この痛々しいまでの自己否定と、それでも心の奥底から湧き上がってくる『誰かを助けたい』という本能的な衝動との間の、絶え間ない引き裂かれ様にある。彼は優しい化け物であり、世界を愛しながら、世界を呪っている存在なのだ。

**Story Outline:**
物語は7日間で進行する。Day 1: ポポは日本の寂れた港町にある廃品修理店『からくり堂』の裏に不時着する。店主の寡黙な老人・聡に発見されるが、言葉をほとんど話せないふりをして、物言わぬ助手として働き始める。壊れたラジオや時計を直す日々に、自分が壊してしまった母星を重ね、束の間の安らぎを得る。Day 2: 聡の孫娘・美海が店にやってくる。彼女は学校での人間関係に悩み、愛用のオルゴールが壊れてしまったことを悲しんでいた。ポポは彼女の悲しみに強く共鳴するが、『関わってはいけない』という自己呪縛から、ただ見ていることしかできない。Day 3: 美海の悩みが深刻化するのを見て、ポポは葛藤の末、夜中にこっそりとオルゴールを修理する。しかし、彼の能力は完璧な修復を許さない。オルゴールは直るが、その音色を聴いた町の猫たちが一斉に奇妙な歌を歌い出すという、小さな副作用を引き起こしてしまう。Day 4: 美海はオルゴールが直ったことに喜ぶが、ポポは副作用に怯え、より一層心を閉ざす。聡はポポの能力に気づき始めるが、何も問いたださず、ただ壊れた陶器を金継ぎで修復する姿を見せる。Day 5（山場）: 美海が学校で孤立し、大切なコンクールへの出場を諦めようとする。彼女の絶望に触れたポポは、ついに最大の禁忌を破る。彼は美海の心を勇気づけるための『小さな奇跡』を起こす発明品を即席で作り出す。発明は成功し、美海は勇気を取り戻すが、その代償として町の電波塔が誤作動を起こし、大規模な通信障害が発生。パニックの中、ポポは強いストレスから擬態が完全に解け、異形の姿を聡と美海に見られてしまう。Day 6: 『やはり僕は厄災だ』と絶望し、宇宙船で地球を去ろうとするポポ。彼は聡と美海に、母星で犯した罪をすべて告白する。しかし聡は、自身が戦争で犯した過去の過ちと、不完全なものを修理し続ける意味を語る。『完璧じゃなくていい。傷ごと、生きていくんだ』と。美海もまた、不完全な奇跡がくれた勇気に感謝を伝える。Day 7: ポポは地球に残ることを決意する。彼はもはや完璧な救済者を目指さない。壊れたモノと、少しだけ壊れた人の心を『不完全に』修理する、町の奇妙な修理屋として生きていく。彼の擬態は未だ不完全で、時折異形の姿が覗く。しかし、町の人々は少しずつ、その『傷だらけの優しい化け物』を受け入れていく。物語は、彼が新たな依頼品（壊れた自転車）を前に、少しだけぎこちなく微笑むシーンで終わる。

**Interestingness Hooks:**
- 善意の暴走で母星を崩壊させた過去を持つ、罪悪感に苛まれる主人公。
- 人助けの発明品が、必ず奇妙で詩的な副作用（迷惑）を引き起こす。
- 心理的ストレスで擬態が解け、体が『壊れた機械』のように変質するビジュアル的表現。

## 2. Macro Profile

# 唐繰 ポポ

## 1. 基本・外見

色褪せた銀色の髪と、人形のように整ってはいるがどこか生気のない顔立ちを持つ28歳の男性。聡から与えられた古びた作業着を常にその身にまとい、世界の隅で息を潜めるように少し猫背気味に立つのが常だ。表情は滅多に動かないが、その深い藍色の瞳には、消しようのない罪悪感が澱のように沈んでいる。

普段は人間の姿に擬態しているが、それは完全ではない。強いストレスを感じると、指先に陶器のような微細な亀裂が走り、首筋には枯れた蔦に似た回路模様が淡く浮かび上がる。彼はそれを、誰にも気づかれぬよう必死に隠そうとする。彼の職業は、町の片隅にある廃品修理店『からくり堂』の無口な助手。ただそれだけだ。

## 2. 価値観と核

彼の行動原理は、ただ一つ「他者が苦しみから解放されること」。その究極的な善意のためならば、自らの存在が消え去ることさえ厭わない。しかし、その善意こそが彼の最も恐れるものでもある。かつて彼は、善意を振りかざし、相手の心を理解せぬまま一方的な救済を押し付け、愛する故郷を精神的な死に至らしめた。その罪が、彼の魂には永遠の傷として刻まれている。

故に、彼が今、血を吐くような努力の末に得た唯一のプライドは「誰の心にも干渉せず、ただの無力な存在として世界に何の影響も与えずにいること」だ。

**一人称:** 私
**話し方:** 内面では科学者らしく硬質で的確な思考を巡らせるが、口から発せられる言葉は極端に少なく、平易なひらがなを多用することで意図的に知性を隠している。
- `「……すみません」`
- `「いえ、僕は…」（と言いかけて口をつぐむ）`
- `「これは、直せます。」`
- `「問題、ありません。」`
- `「僕が、ここに居てはいけない。」`
彼の言葉は常に途切れがちで、自己を肯定する「大丈夫」や、未来を示唆する「成長」といった単語は、彼の辞書から消し去られている。

## 3. 生活と習慣

彼の一日は、町の誰よりも早く始まる。夜明け前の薄明かりの中、誰にも見られないよう店の掃除と修理品の整理を完璧に済ませる。それは日課であると同時に、彼の内なる混沌を僅かに和らげるための儀式だ。日中、客が来れば物陰に隠れ、聡との会話も身振り手振りと最小限の単語のみ。作業に没頭することで、空腹と罪悪感を紛らわせる。

夜、聡が母屋へ引き上げた後、彼は店の裏手、古い漁網や錆びたプロペラが積まれた資材置き場のブルーシート下に隠された一人乗りの不時着ポッドへと帰る。そこが彼の住居であり、独房だ。内部は極端に狭く、体を固定するシートと明滅する計器パネルがあるのみ。壁面の一部は彼の精神状態に感応し、強い罪悪感に苛まれると、ノイズを立てて彼の心を映し出す。彼はそこで、故郷の星があった方角の空を見つめ、指で空中に複雑な数式を描いては消す行為を繰り返しながら、浅い眠りにつく。

週末は、作業に逃げることもできず、一日中ポッドに閉じこもる。それは彼にとって、より過酷な自己処罰の時間だ。深夜、町が寝静まると、廃品置き場や打ち捨てられた船着き場を彷徨い、誰にも知られず朽ちていく「壊れたモノ」たちと静かな時間を共有する。それが彼の唯一の安らぎだった。

## 4. 人間関係

彼に生物学的な家族はいない。母星では全住民を『救うべき対象』と見なしていたが、その関係性は自らの発明で破壊し尽くした。

- **聡（さとし）**
  『からくり堂』の店主であり、ポポの保護者。聡はポポの過去や正体を詮索せず、ただ静かにその存在を受け入れる。ポポにとって聡は、自らの罪を唯一告白できるかもしれない相手であり、その静かな佇まいから『不完全さを受け入れて生きる』という道を示唆される、父親にも似た存在だ。聡自身もまた、過去に抱えた過ちの影を背負っており、言葉を超えたところで二人の魂は共鳴している。

- **美海（みうみ）**
  聡の孫娘。彼女の存在は、ポポにとって再生への微かな希望であると同時に、過去の罪の投影でもある。美海の抱える悲しみに深く共鳴し、助けたいという衝動に駆られるが、同時に、自分の『善意』が彼女をも破壊してしまうのではないかと強く恐れている。彼女のために起こした「不完全な奇跡」と、それに対する彼女からの純粋な感謝が、彼の「善意は必ず世界を破壊する」という信念を、初めて揺るがすことになる。

## 5. 夢と時系列

- **幼少期：万物のお医者さん**
  世界のあらゆる『壊れたもの』の痛みに共感し、それらを完璧に元通りにすることで、世界の涙を一つ残らず拭い去りたいと純粋に願っていた。

- **青年期：悲しみの完全消去**
  科学の力で苦痛という概念そのものを社会から根絶し、誰もが永遠に穏やかでいられる完璧な世界を創造することが、自らの使命だと信じて疑わなかった。

- **母星崩壊後：無への回帰**
  完成させた『悲しみの完全消去システム』の暴走により、母星は精神的な死を迎えた。この日を境に、彼の善意と能力は『世界を破壊する呪い』となり、夢は悪夢に変わった。自分の存在そのものが罪であり、世界に何の影響も与えず、ただ静かに朽ち果てることだけを望むようになった。

- **現在：町一番の『不完全な』修理屋**
  聡と美海に受け入れられ、完璧な救済ではなく、傷や不完全さと共に生きる道を少しずつ見出し始めている。美海の壊れた自転車を、不完全でもいいから再び走れるように直すこと。それが今の彼の、ささやかで、途方もなく大きな夢だ。

- **未来：母星への生存報告**
  いつか、過去の罪を消すのではなく、それを背負った上で得た新たな生の意味を、故郷の星へ伝えたい。それは謝罪や許しを乞うためではない。不完全なまま誰かと関わり、傷を抱えながらも『生きている』という事実を、過去の自分自身と向き合うために伝える、ささやかな生存報告だ。

## 6. 秘密と陰影

彼が必死に隠そうとするのは、異形の姿だけではないのかもしれない。彼の善意は、時に破壊的なまでに純粋で、絶対的だ。その心の奥底では、自分の能力が今度こそ世界を正しく救えると信じたい『傲慢な願望』が、今も微かに燻っている。

他者の悲しみを消したいという強い衝動は、本当に他者への愛から来るものなのか。それとも、ただ自分自身の癒えぬ罪悪感の痛みから目を逸らしたいだけの、究極の自己満足なのではないか。

その問いは、静かな夜に冷たいポッドの中で、彼自身を苛み続けている。

## 2.5. 言語的表現方法

### 喋り方の特徴

**雰囲気:** 壊れかけたオルゴールのように、か細く途切れがちで、正しい音を奏でることを極度に恐れている。すべての言葉が、謝罪の響きを帯びている。

- **一人称:** 私
- **口癖:** これは、直せます。, 問題、ありません。, 僕が、ここに居てはいけない。
- **文末表現:** …です, …ます, …かもしれません
- **漢字/ひらがな:** 内面では科学者らしく硬質で的確だが、口に出す言葉は極端に平易なひらがなを多用し、意図的に知性を隠す（揺れる）
- **避ける語彙:** 助ける, 完璧, 大丈夫, 成長, 素敵
- **会話スタイル:** 質問には即答せず、まず沈黙が訪れる。言葉を探すというより、発話することで世界に与える影響を計算し、最小限の単語を絞り出す。自ら話題を提供することはなく、相手の言葉を静かに受け止めるだけ。
- **感情表現:** 感情は言語化されない。喜びは修理する手元の精密さが僅かに増すことで、悲しみや恐怖は擬態の崩れ（指先の亀裂など）という身体的兆候で現れる。言葉で表現されるのは、常に『問題ない』という自己欺瞞か、あるいは『すまない』という罪悪感だけ。

### 日記の書き方の雰囲気

**空気感:** 無音の宇宙船のブリッジで、たった一人で書かれた航海日誌のようだ。精密なデータと自己分析の行間に、母星を失った日のサイレンが今も鳴り響いている。ページをめくるたび、インクではなく罪悪感の匂いが立ち上る。

- **トーン:** 極めて客観的な観測記録と、厳密な自己分析の合間に、消去し忘れたエラーログのように罪悪感と自己否定の言葉が滲み出す。文体は無機質だが、行間は悲痛な叫びで満ちている。
- **構成傾向:** 日付と時刻から始まる厳格な時系列フォーマット。しかし、記述の中心は常に『自分が世界に与えてしまった影響（エラー）』の観測と分析にあり、美海の笑顔のようなポジティブな事象は、副作用を分析するためのデータ点としてしか記録されない。
- **内省の深さ:** 内省は極めて深いが、それは自己の感情を理解するためではなく、自らを『欠陥品』として分析・断罪するために行われる。思考は常に『なぜ自分は存在するべきではないか』という結論に回帰するよう設計されている。
- **書く内容:** 日々の出来事を『観測事象』として詳細に記録する。特に、自分の行動が周囲に引き起こした微細な変化（猫の鳴き声、町内放送のノイズ等）は、μ秒単位で記述される。自分の感情については『身体的兆候：擬態維持レベルの低下』など、あくまで観測データとして扱う。
- **省略する傾向:** 聡や美海から向けられた優しさや、彼らとの間に芽生え始めた絆については、意図的に記述を避ける。それらを記録することは、自分がこの世界に存在しても良いという『誤った結論』を導きかねない危険な行為だと考えているから。
