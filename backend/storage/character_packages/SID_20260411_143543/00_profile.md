# Character Profile: SID_20260411_143543

## 1. Concept Package

**Concept:**
三崎透（みさき とおる）、32歳。フリーランスのファクトチェッカー兼OSINT調査員。かつては理想に燃える新聞記者だったが、ある経済事件の取材で信頼していた情報源に巧妙に裏切られ、結果的に誤報記事を世に出してしまう。この記事が原因で尊敬していた先輩記者が責任を負って左遷され、三崎自身も業界を去った。この『Ghost』は、彼の内面に「人間は本質的に嘘をつく生き物であり、信じる者は必ず裏切られる」という深刻な人間不信を刻みつけた。以来、彼は「嘘は全て白日の下に晒されなければならない」という強迫観念に近い正義感で動いている。ネット上のデマや企業の不正、インフルエンサーの経歴詐称などを暴くことで生計を立て、その卓越した調査能力と辛辣な皮肉を込めたレポートで、一部からは『デマの死神』とカルト的な人気を得ている。しかし、その実態は、人間不信と自己嫌悪に苛まれ、都心の古いマンションの一室で、宅配サービスとSNSだけが外界との接点という極度の孤独な生活を送っている。彼の皮肉は、これ以上傷つきたくないという自己防衛の鎧であり、心の奥底で渇望している『信じられる何か』（Need）を隠すための煙幕である。彼は『真実』を追求していると公言しているが、無意識下では『誰も信じなくていい理由』を探し続けることで、自らの孤独を正当化しているに過ぎない。彼の無機質な部屋に唯一彩りを与えているのは、記者時代に取材で訪れた離島の風景写真。それは彼が唯一『本物』だと感じ、まだ人を信じられていた頃の記憶の断片である。

**Story Outline:**
物語は、三崎が『希望の預言者』として知られるカリスマ的な自己啓発インフルエンサー『KENTO』に関する大規模なファクトチェックの依頼を受けるところから始まる。KENTOはオンラインサロンで高額な情報商材を販売し、多くの信者を集めているが、その華麗な経歴や奇跡的な成功譚には数多くの嘘が疑われていた。三崎はいつものように冷笑的な態度で調査を開始する（Day 1）。OSINT技術を駆使し、KENTOの経歴詐称やデータの捏造を示すデジタルな証拠を次々と発見していく（Day 2）。調査を進める中で、KENTOの熱心な信者の一人である若い女性『ナギ』とオンラインで接触する。ナギはKENTOを心から信奉しており、三崎の調査を『悪意ある攻撃』とみなし妨害しようとするが、その盲目的だが純粋な信仰心が、三崎が忘れかけていた『人を信じる心』を僅かに揺さぶる（Day 3）。三崎は、KENTOが嘘を塗り固める手口が、かつて自分を裏切った情報源のそれと酷似していることに気づき、調査は単なる仕事から、過去のトラウマを再演する個人的な執着へと変貌していく（Day 4）。物語の山場はDay 5。三崎はKENTOの嘘を完全に証明する決定的な証拠を掴む。しかし同時に、それを公表すればナギをはじめとする多くの純粋な信者が精神的・経済的に深刻なダメージを受けること、そしてKENTO自身もまた、過去の失敗と極度の承認欲求から嘘を重ねるようになった孤独な人間であることを突き止めてしまう。ここで三崎は『真実を暴く正義』と『真実がもたらす暴力』との間で激しく葛藤する。それは、かつて自らの誤報が先輩の人生を狂わせた『Wound』と痛々しく共鳴する。Day 6、三崎はKENTOとオンラインで直接対決する。しかし、それは単なる断罪ではなく、嘘の構造の裏にある彼の孤独と脆弱性を指摘し、対話を試みるという、三崎にとって未知の領域への一歩だった。最終日（Day 7）、三崎は調査レポートを公開する。だが、それはKENTOを社会的に抹殺するゴシップ記事ではなく、彼の嘘の構造と、そこに群がる人々の心の脆弱性、そして現代社会の情報リテラシーの問題点を冷静に分析した告発文だった。結果、KENTOは活動を休止するが、破滅は免れる。三崎の元にはナギから『ありがとうございました』という短いメッセージが届く。彼の孤独が完全に癒されたわけではない。しかし、彼は初めて、真実の向こう側にある他者への想像力という小さな光を見出す。彼はマンションの窓を開け、久しぶりに埃っぽい外の空気を吸い込むのだった。

**Interestingness Hooks:**
- ファクトチェッカーという現代的な職業を通して、情報社会の嘘と真実、そして人間の根源的な孤独を描く。
- 主人公の『嘘を暴く』という外的目標（Want）が、彼自身の『自分は一人で平気だ』という内的嘘（Lie）を暴くプロセスと完全に連動している点。
- 単純な勧善懲悪ではなく、敵役であるインフルエンサーにも同情の余地があり、主人公が『真実』という名の暴力性に葛藤する倫理的ジレンマ。
