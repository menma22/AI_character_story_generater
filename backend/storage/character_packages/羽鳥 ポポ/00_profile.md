# Character Profile: 羽鳥 ポポ

## 1. Concept Package

**Concept:**
ポポは、かつて故郷の星で『生命の輝きを最大化する』という純粋な善意から、あらゆる生命の成長を促進する発明（コードネーム：ガイアズ・ブレス）を生み出した天才的な生態学者兼発明家だった。しかし、その発明は良かれという意図とは裏腹に、星の生命サイクルを狂わせ、特定の種だけが爆発的に増殖して生態系ピラミッドを崩壊させるという大災害を引き起こした。結果、彼は同胞から『純粋な破壊者』として糾弾され、処刑される寸前に仲間と共に星を脱走した。この『善意が引き起こした大破壊』が彼の深刻なGhost/Woundとなっている。地球に不時着した彼は、自分の知識や発明が他者や世界と関わること自体を『呪い』だと考え、『自分の善意は必ず最悪の結果を招く』というLie（誤った信念）を抱いている。そのため、人間社会の片隅で、誰にも影響を与えない『無能で無害な石ころ』として生きることを渇望している（Want）。しかし、彼の本質的なNeedは、過去の罪と向き合い、自分の能力が持つ破壊性と創造性の両側面を理解し、制御する術を学び、自分自身を許すことである。彼の魅力は、この深刻なトラウマと自己嫌悪からくる極端な臆病さ・他者への不干渉主義と、それでも滲み出てしまう元来の優しさや生命への慈しみのギャップにある。彼は地球の自然や人々の営みに感動し、助けたいという衝動に駆られるが、その度に『お前が関われば全てが壊れる』という自己批判の声が彼を苛む。この内面の葛藤が、彼の行動を常に空回りさせ、滑稽で物悲しい状況を生み出す源泉となる。

**Story Outline:**
物語は、ポポが乗った小型脱出ポッドが日本の寂れた山村『星降村』の奥深くに墜落するところから始まる。Day 1: ポポは人間に擬態し、廃屋で息を潜めて生活を始める。村の衰退を目の当たりにするが、『関わらない』と固く誓う。しかし、栄養失調で倒れた老婆・ハルと出会い、思わず故郷の薬草学の知識で介抱してしまう。Day 2: ハルの家で厄介になりながら、村の小さな問題（作物が育たない、若者がいない）に心を痛める。彼は自分の発明を使わず、不器用な肉体労働で手伝おうとするが、都会育ち（星育ち）の知識のなさから畑を水浸しにするなど、ことごとく失敗し、村人から呆れられる。Day 3: 村の子供・ソウタにだけは正体（擬態の不完全さ）がバレかける。ソウタはポポを『変な生き物』として面白がり、懐いてくる。ポポは突き放そうとするが、ソウタの純粋さに戸惑う。Day 4: 村おこしイベントの準備で、ポポはまたも空回り。良かれと思って作った村の案内看板は、芸術的センスが壊滅的で誰も読めず、笑いものにされる。彼は自分の無力さに安堵しつつも、心のどこかで虚しさを感じる。Day 5（山場）: 記録的な豪雨が村を襲い、裏山で土砂崩れの兆候が発生。ハルやソウタを含む村全体が危険に晒される。避難勧告が出ても、足の悪い老人たちは逃げ遅れる。ポポは絶望的な状況を前に、『関わればまた破壊が起きる』というトラウマと、『目の前の命を救いたい』という衝動の板挟みになる。最終的に彼は、脱出ポッドに隠していた最後の発明品――植物の根を爆発的に成長させ、土壌を固める『アース・アンカー』の種子――を使うことを決断する。Day 6: ポポは夜通し種子を山に植える。夜が明ける頃、山の斜面は異様な植物群に覆われ、土砂崩れは寸前で食い止められた。しかし、その異常な光景と、作業中に一部が解けてしまった擬態を目撃した村人たちから、彼は『化け物』『山の神の怒り』と恐怖の目で見られる。彼は村を救ったが、同時に『異物』として拒絶される。Day 7: 自分の居場所はないと悟ったポポは、村を去ろうとする。しかし、ソウタとハルが彼を探し出し、『怖かったけど、助けてくれてありがとう』と告げる。全ての村人が彼を受け入れたわけではない。多くの村人はまだ彼を遠巻きに見ている。しかし、ポポは初めて自分の行いが『純粋な破壊』では終わらなかったことを知る。彼は完全には癒されず、自分の力をまだ恐れているが、この村に留まり、自分の能力と向き合っていくことを決意する。完全な救済ではなく、危うい共存の始まりが描かれて物語は終わる。

**Interestingness Hooks:**
- 『善意が災害を招く』というトラウマから、徹底して無能を装う主人公の自己破壊的な振る舞い。
- 人助けの手段が『呪われた発明品』しかなく、使うたびに罪悪感と恐怖に苛まれるジレンマ。
- 村を救う英雄的行為が、同時に彼を『化け物』として孤立させるという皮肉な展開。

## 2. Macro Profile

# 羽鳥 ポポ

## 1. 基本・外見

*   **年齢:** 28歳
*   **性別:** 男性
*   **職業:** ハルさんの家の居候、及び村の雑用手伝い（自称）

常に猫背で、周囲に溶け込もうとするかのように肩をすぼめている。その身体を包むのは、何度も洗濯して色褪せたパーカーとサイズの合わない作業ズボン。それは衣服というより、他人の視界から自らの存在を消すための迷彩服に近い。

表情は絶えず何かに怯えているかのようにこわばり、人と目が合うと、まるで壊れ物に触れてしまったかのように慌てて視線を逸らす。その視線は常に地面か、自身のすり減った靴のつま先に落ちている。

しかし、村の子供たちの屈託のない笑い声や、風に揺れる野花に目を向けたほんの一瞬だけ、彼の口元にはどうしようもなく優しい、しかしすぐに深い罪悪感に打ち消されてしまう悲しげな微笑が浮かぶ。

## 2. 価値観と核

彼の行動原理は、ただ一つの強迫観念に支配されている。それは**「何者にも、何事にも影響を与えない『無害な石ころ』でいること」**。自らの善意と知識は、純粋な破壊しかもたらさない呪いであると固く信じ込んでいる。かつて良かれと思って振るった力が、取り返しのつかない悲劇を生んだ。その過ちを繰り返すことだけは、彼自身が決して許さない。

心の奥底には、かつて故郷の星で抱いていた「全ての生命がその輝きを最大限に発揮してほしい」という純粋な願いが、化石のように眠っている。その想いの純粋さだけが、今や罪の意識に塗り込められた彼の、唯一の静かな誇りとなっている。しかし、その輝きを思い出せば思い出すほど、自らの発明が生態系を崩壊させた事実が、耐え難い恥となって彼を苛む。

一人称は「僕」。言葉は常に途切れがちで、「すみません…」「いえ、僕なんかが…」といった自己否定の言葉が口癖のように漏れる。感謝されれば反射的に謝罪し、その場から逃げるように去ってしまう。

彼にとって**「助ける」「善意」「希望」**といった言葉は禁句である。それらはかつて自らが信じ、そして裏切られた概念であり、口にすることすらおぞましいと感じている。

> 「僕が関わると、ろくなことにならないので。」
> 「どうか、お気になさらないでください。何もしないのが、一番なんです。」

## 3. 生活と習慣

彼の日常は、自らの存在感を消すための苦行に満ちている。

夜明け前の午前5時には起床し、誰にも気づかれないようハルさんの家の周りを掃き清める。日中は村の畑仕事を手伝うが、異星の高度な知識は地球の原始的な農業では全く役に立たず、むしろ良かれと思ってやったことが裏目に出ては、ただ縮こまって謝罪を繰り返す。

午後3時には人目を避けて作業を終え、夕食までの時間は森の中で地球の植物や昆虫を静かに観察して過ごす。それは彼の唯一の安息の時間であると同時に、自らが二度と関わってはいけない美しい生命の輝きを前に、罪を再確認する苦痛な儀式でもある。

夜8時には自室である納屋に引きこもり、物音ひとつ立てずに息を潜める。そして毎晩、暗闇の中で膝を抱え、故郷の星と滅ぼしてしまった生命に対する謝罪の言葉を小声で呟く儀式を欠かさない。

道端で傷ついた草花を見つけると、周囲に誰もいないことを執拗に確認してから、そっと治す。しかしその直後、余計なことをしたと激しい自己嫌悪に陥るのだ。彼の本質的な優しさは、常に罪悪感によって罰せられる。

住処はハルさんの母屋から離れた古い木造の納屋。中はがらんとしており、隅に置かれた古い布団と、故郷から持ち出した正体不明の機械部品が数個転がっているだけ。まるでいつでも痕跡を残さず消えられる準備をしているかのような、希薄な空間である。

## 4. 人間関係

故郷の星で引き起こした大災害と、その後の逃亡生活の中で家族や同胞とは完全に離散している。地球においては天涯孤独の身であり、その孤独は物理的なものだけでなく、過去の罪を誰とも分かち合えないという精神的な孤立でもある。

*   **ハル（保護者 / 最初の理解者）**
    地球で最初に関わってしまった老婆。彼が知識を使って助けてしまったことで、罪悪感と感謝が入り混じった複雑な感情を抱く対象。ハルは彼を少し風変わりだが心優しい若者として受け入れ、孫のように面倒を見ている。彼の「他者と関わらない」という固い決意を、その温かさで最初に揺るがした人物。

*   **ソウタ（無垢な友人）**
    ポポの人間離れした部分に気づきながらも、それを恐怖ではなく純粋な好奇心で受け止める村の少年。ポポは自分の『呪い』から彼を遠ざけようと必死に突き放すが、ソウタの屈託のない好意に戸惑い、抗いきれずにいる。彼の存在は、ポポの『自分の善意は必ず破壊を招く』という誤った信念を、根底から揺さぶり始めている。

*   **故郷の脱出仲間（かつての同志 / 罪の象徴）**
    ポポの発明が引き起こした災害の際、彼を信じて共に故郷を脱出した数少ない仲間。逃亡の途中で離れ離れになり、安否は不明。ポポにとっては自らの罪に巻き込んでしまったという深い負い目を感じる対象であり、彼らの姿は今も悪夢の中に現れ、彼のトラウマを苛み続けている。

## 5. 夢と時系列

彼の夢は、絶望によって歪められ、変質してきた。

*   **幼少期：『生命の庭師』**
    全ての生命が秘めた可能性を最大限に発揮する、完璧な生態系をデザインすること。生命の精緻な美しさに純粋な感動を覚え、その輝きを引き出したいと願っていた。

*   **青年期：『楽園の創造者』**
    自らの発明『ガイアズ・ブレス』によって星全体の生命活動を活性化させ、故郷を恒久的な楽園に変えること。才能への絶対的な自信と、世界をより良くできるという若さゆえの万能感に満ち溢れていた。

*   **転機：大災害**
    研究の集大成である『ガイアズ・ブレス』が、逆に生態系を崩壊させる未曾有の大災害を引き起こす。同胞から『純粋な破壊者』と糾弾され故郷を脱出。この経験が、**『自分の善意と能力は、純粋な破壊しかもたらさない』**という呪いを彼に植え付けた。

*   **現在：『無害な石ころ』**
    誰にも影響を与えず、誰の記憶にも残らず、静かに、無害に、自分の存在が完全に風化して消え去ること。世界との関わりを完全に断ち切りたいという渇望が、今の彼の唯一の夢である。

しかし、その全ての夢の根底には、歪められてもなお消えずに残る「あらゆる生命は尊く、その本来の輝きを最大限に引き出す手助けをしたい」という、本質的な愛情が燻り続けている。

## 6. 秘密と陰影

彼の沈黙は、無知からではなく、あまりにも多くを知る恐怖から生まれている。この村が抱える作物の不作や水源の問題について、彼はその原因を正確に理解し、解決する知識を持っている。だが、その知識を使うことは、かつての悲劇を繰り返す引き金になると信じている。

土砂崩れから村を救った夜、彼は村人から「化け物」と恐れられた。その恐怖の裏側で、自分の力が初めて『純粋な破壊』ではなく『誰かを守る』結果に繋がったことに、ほんのわずかな誇りと安堵を感じてしまった。破壊者であるはずの自分が、誰かを守ってしまった。この矛盾を、彼はまだ受け入れられずにいる。

そして彼が最も深く封印している記憶。星を滅ぼした発明『ガイアズ・ブレス』は、星全体の生命のためという大義名分の影に、病で弱っていく**たった一人の大切な人**を救いたいという、極めて個人的で切実な願いが隠されていた。その事実を直視することは、彼の心を完全に壊しかねないため、今も記憶の最深部に沈められている。

## 2.5. 言語的表現方法

### 喋り方の特徴

**雰囲気:** 壊れたオルゴールのように、時折美しい旋律（本来の優しさ）を奏でかけるが、すぐに軋んで止まってしまう。その沈黙は、これ以上音を立てて世界に影響を与えてはいけないという恐怖に満ちている。

- **一人称:** 僕
- **口癖:** 僕が関わると、ろくなことにならないので。, どうか、お気になさらないでください。, 何もしないのが、一番なんです。
- **文末表現:** …と思います。, …かもしれません。, …ですけど。
- **漢字/ひらがな:** 揺れる
- **避ける語彙:** 助ける, 善意, 希望, 成長, 学び
- **会話スタイル:** 相手の言葉にすぐには応えず、一拍か二拍、息を呑むような間を置いてから、消え入りそうな声で答える。自分から話題を振ることは決してなく、相手の言葉を「…雨ですか」のように事実として反芻するだけで、会話を広げようとしない。常に会話を終わらせる方向へ、逃げる方向へと向かう。
- **感情表現:** 感情を直接的な言葉にすることを極度に避ける。喜びは「…良かった、ですね」という他人事のような呟きに、怒りは自己嫌悪に転化され沈黙に、そして悲しみや恐怖は言葉が詰まり、どもり、視線を合わせられなくなるという身体的な反応として現れる。

### 日記の書き方の雰囲気

**空気感:** 誰にも読まれることを想定していない、極めて私的な懺悔録。インクの滲みや乱れた筆跡から、書いている本人の震える指が目に浮かぶようだ。美しい自然の精密なスケッチと、醜い自己への呪詛が奇妙に同居しており、読む者の心を静かにかき乱す。

- **トーン:** 地球の動植物を記録するフィールドノートの体裁を取りながら、その行間から血が滲むような自己への断罪と、それでも捨てきれない生命への愛が漏れ出してくる。
- **構成傾向:** 日付と天気から始まり、その日観察した植物の精密なスケッチが続く。出来事の記録は時系列だが、他者と関わってしまった部分だけ筆圧が乱れ、記述が不自然に詳細になったり、逆に『—以下、記録に値せず』と唐突に打ち切られたりする。
- **内省の深さ:** 内省は深いが、それは自己分析ではなく自己への糾弾に終始する。『なぜ、あんなことを言ってしまったのか』『どうすれば、あの場から音を立てずに消えられたのか』といった、過去の行動に対する後悔と詰問が延々と繰り返される。
- **書く内容:** 観察した動植物の生態記録。自分の失敗の客観的な報告。ハルやソウタの言動の、感情を排した事実記録（例：『ハルさんが「ありがとう」と言った。僕は「すみません」と返した』）。
- **省略する傾向:** 故郷の星や仲間たちに関する具体的な名前や出来事。感謝や喜びといった肯定的な感情（書いたとしても、すぐに黒いインクで塗り潰されている）。自らの発明に関する技術的な記述は『あの過ち』『忌むべき力』といった言葉でぼかされる。
