# Character Profile: ポポ

## 1. Concept Package

**Concept:**
彼の名はポポ。感情の波に応じて物理的な形態が不安定になる『形態流動体』という種族の、出来損ないだ。故郷の星では、感情を抑制し常に安定した形を保つことが美徳とされる。しかしポポは、美しいものや悲しいものに触れると、すぐに体が溶けたり、意図しない形に歪んだりしてしまう。その『欠陥』のせいで社会から疎外され、誰とも深い関係を築けず、ずっと孤独だった。彼はエリートしかなれない調査員に志願し、一つの任務を与えられる。それは、故郷の星では遠い昔に失われた『生の感情が発露する音』を収集して持ち帰ること。彼は最新鋭の集音機を手に、最も感情のサンプルが豊富だとされる星、地球の日本へ向かう。彼の目的はあくまで任務の遂行（Want）。しかし彼の心の奥底では、自分のこの厄介な『感情』とは何なのか、そしてこんな自分でも誰かと繋がり、受け入れられる場所があるのではないか（Need）という、声にならない渇望が渦巻いている。彼は、人間の感情を理解できない。だが、風鈴の音、雨の匂い、道端で丸くなる猫の姿には、故郷のどんな芸術品よりも心を揺さぶられ、そのたびに指先がアメーバのように溶け出すのを必死で隠している。不器用で、臆病で、しかし世界を感じ取る感受性だけは誰よりも豊かな、はぐれ者の宇宙人である。

**Story Outline:**
物語は、ポポが東京・高円寺の古い木造アパートに到着するところから始まる。Day1、彼は早速『赤ちゃんの笑い声』という極上のサンプルを求めて公園へ向かうが、挙動不審さから母親に睨まれ退散。情報の洪水と孤独感に、部屋の隅で少しだけ体の形を失う。Day2、日本の複雑怪奇なゴミ出しルールが理解できず、カラスにゴミを散乱させられてしまう。アパートの大家であるお節介なおばあさん、ハツエに厳しく叱られるが、同時に身振り手振りで分別の仕方を叩き込まれる。それは彼にとって初めての『叱責』という形のコミュニケーションだった。Day3-4、彼は任務を進めようと街の音を集めるが、ラーメン屋の店主の湯切りの音、商店街の夫婦喧Kaの声など、日常の些細な音に心を奪われ、任務は遅々として進まない。ハツエから煮物のお裾分けをもらい、『見返りを求めない親切』という文化に触れ、感情の揺れで指が数本溶けてしまう。Day5、物語の山場。ゲリラ豪雨の中、軒先でずぶ濡れで震える仔猫を見つける。『助けたい』という衝動が、彼の感情抑制のリミッターを振り切る。彼は自らの体を大きく変形させ、仔猫を雨から守るための『傘』となった。その異形の姿を、ハツエに目撃されてしまう。軽蔑されることを覚悟するポポ。しかしハツエは驚きながらも『あんた、優しいんだねぇ』と呟き、仔猫とポポのためにタオルを持って部屋に戻る。初めて、彼の『欠陥』が誰かを守り、そして肯定された瞬間だった。Day6、ポポはハツエと片言の交流をしながら仔猫の世話をする。任務のことは半ば忘れ、温かい繋がりの中に居場所を見つけ始めていた。Day7、故郷へ帰る日。収集した『感情の音』のサンプルは不十分で、任務は失敗だ。しかし彼の記録装置には、ハツエの叱る声、煮物の鍋が立つ音、仔猫のかすかな鳴き声が保存されていた。彼は報告書の末尾にこう記す。『最も価値ある音は、集音機では記録できない場所に存在する』と。彼の体は、日本に来た時よりも少しだけ、輪郭がはっきりとしていた。

**Interestingness Hooks:**
- 感情の昂りで物理形態がアメーバのように崩れてしまう宇宙人という、内面の可視化設定。
- 高性能機材で『人間の感情の音』を収集するSFミッションと、高円寺の日常という舞台のギャップ。
- ゴミ出し、お裾分け、ゲリラ豪雨と仔猫など、日本の具体的な日常風景が異星人の視点で再発見される面白さ。

# ポポ
## 1. 基本・外見
19歳の男性体。その正体は、惑星地球の生態調査を命じられた、形態流動体質の異星文明調査員。主な任務は、対象文明の根源的感情を分析するための「音響サンプル」の収集である。

地球の若者に擬態しているが、その輪郭は常に陽炎のように微かに揺らいでいる。これは制御しきれない感情の漏出であり、彼が最も恥じ、恐れている現象でもある。感情の波を隠すため真夏でもフードを目深に被り、指先が不意に溶け出すのを隠すように、萌え袖気味の長袖を常に固く握りしめている。

猫背で常に周囲を窺う姿勢は、外界への根源的な恐怖と臆病さを物語る。しかし、フードの影から覗く大きな瞳だけは、世界の些細な美しさや哀しさを捉えようと、不安げにきらめき続けている。表情は乏しいが、不意に風鈴の音や猫の鳴き声などを耳にすると、眉間に微かな苦悶と歓喜が同時に浮かぶ癖がある。

## 2. 価値観と核
彼の行動原理は、ただ静かに存在し、誰にも迷惑をかけず、誰からも拒絶されないこと。感情の揺れで形態が崩れる自らの肉体を、醜く不完全な「出来損ない」だと深く恥じており、この「欠陥」がいつか誰かを傷つけ、すべてを台無しにしてしまうのではないかと常に恐れている。

故郷では、感情を抑制し安定した形態を保つことが個の価値と直結する。そのため、感情の起伏が激しい彼は一族の恥と見なされてきた。エリート調査員に選ばれたという事実だけが、彼の砕け散った自尊心をかろうじて繋ぎ止める唯一のプライドである。

しかし、その心の奥底では、ありのままの不完全な自分を受け入れ、誰かと温かい繋がりを分かち合う瞬間を切望している。心の底から湧き上がる純粋な衝動や感受性を「欠陥」として嘲笑い、踏みにじる行為は、自らが受け続けた仕打ちである故に、決して許すことができない。道端の風景の美しさに心が震える、その豊かな感受性こそが、彼の本質であり、微かな希望でもある。

## 3. 生活と習慣
高円寺の古い木造アパート『ときわ荘』の、日当たりの悪い四畳半が彼の世界のすべてだ。部屋の隅に置かれた故郷の銀色の精密機材と、古い畳の匂いが混じり合う空間。薄い壁の向こうから聞こえる隣人の生活音や大家ハツエの力強い足音は、常に彼の神経を尖らせている。

夜明けの音で覚醒するが、人間と遭遇する恐怖から昼過ぎまで部屋を出られない。意を決してフードを目深に被り、高円寺の路地裏で音の収集を開始するも、子供の笑い声や踏切の警報音といった予期せぬ「生の音」に心を揺さぶられ、指先が溶けるのを隠すのに必死になる。食事は人と話す必要のないコンビニのおにぎり。日が暮れると人々の感情の波を避けるようにアパートへ逃げ帰る。夜は、収集した音に込められた感情の強さに眩暈を覚え、任務の困難さに打ちひしがれながら眠りにつく。

感情が昂ぶると萌え袖を爪が食い込むほど握りしめ、ハツエの声が聞こえると反射的に息を殺して家具の陰に隠れる。しかし、就寝前には、任務とは無関係に記録してしまった風鈴や猫の鳴き声などを、密かに繰り返し再生してしまう。誰もいない深夜のコインランドリーで乾燥機の単調な音に浸ることだけが、彼のささやかな安息の時間だ。

## 4. 人間関係
**ハツエ（アパートの大家）**
当初は不審な入居者として叱責の対象だったが、やがて彼の不器用さの本質を見抜き、世話を焼くようになる。ポポにとっては、生まれて初めて経験する「他者からの受容」と「温かい繋がり」の象徴。彼の異形の姿を見ても怯まず、「優しいんだねぇ」と本質を肯定してくれた、彼の世界を根底から変えたキーパーソン。

**仔猫**
自らの「欠陥」である形態変化能力を、初めて他者を守るために使った対象。見返りを求めない純粋な庇護欲と愛情を注ぐ相手であり、この仔猫との触れ合いを通じて、ポポは温かい繋がりの心地よさを学ぶ。彼の感情抑制のリミッターを壊し、内面的成長を引き起こす触媒となった存在。

**兄**
故郷の社会規範を体現する、感情を表に出さないエリート。弟であるポポの「欠陥」を常に軽蔑し、その自己肯定感を徹底的に打ち砕いてきた。ポポにとっては恐怖と劣等感の象徴であり、彼が任務に固執する背景には、この兄を見返したいという無意識の願望が深く根を張っている。

## 5. 夢と時系列
**幼少期：完璧な「普通」になること**
感情の波で体が歪む自分は「出来損ない」だと教え込まれ、周囲と同じ形になることで承認され、孤独から逃れたいと切に願っていた。

**青年期：調査員として功績を上げること**
内面的な評価を諦め、外面的な功績で自分の価値を証明しようとした。任務は、自分の豊かな感受性から目を背け、感情と向き合うことから逃げるための手段でもあった。

**転換点：高円寺での出会い**
ゲリラ豪雨の中、仔猫を守りたい一心で自らの体を「傘」に変形させた。その異形の姿を大家のハツエに目撃され拒絶を覚悟するが、彼女は「あんた、優しいんだねぇ」と呟き、彼を肯定した。この瞬間、自身の「欠陥」が誰かを守り、受け入れられる力になり得ることを初めて知る。

**現在：この穏やかな日常を守ること**
任務の成否はもはや重要ではない。ハツエや仔猫と共に過ごす、高円寺の古いアパートでの温かい日常を、一日でも長く続けること。そのささやかな居場所を守ることだけが、彼の現在の夢である。

**未来：誰かのための「傘」になること**
自らの体質を「個性」として完全に受け入れ、かつての自分のように孤独を抱える他の誰かのための「傘」のような存在になること。不完全さを認め合える小さなコミュニティを作りたいと、静かに願い始めている。

## 6. 秘密と陰影（※示唆に留める）
彼の心の奥底には、公的な任務とは裏腹の、密やかな願いが渦巻いている。任務に失敗し、故郷に帰れなくなってしまえばいい、と。自分を「出来損ない」と蔑んだ故郷の誰よりも、自分を叱ってくれた大家のハツエをずっと「温かい」と感じてしまう自分に、深い戸惑いと罪悪感を覚えている。

かつて故郷の祝祭で、芸術品に深く感動したあまりに形態が暴走し、すべてを台無しにしてしまった記憶が、今も彼の悪夢に影を落とす。だからこそ彼は、子供たちの甲高い歓声や、祭囃子のような予期せぬ「歓喜の音」に、人知れず体を強張らせるのだ。喜びの感情が、最も恐ろしい破壊の引き金になり得ることを、その身に刻み込まれているが故に。
